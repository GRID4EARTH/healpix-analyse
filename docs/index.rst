healpix-analyse
===============

**healpix-analyse** is a Python toolkit for analysing HEALPix Earth Observation data.
It provides spherical harmonic transforms, power spectra, gauge-equivariant convolutions,
and multi-resolution up/downsampling operators — all differentiable through PyTorch autograd.
